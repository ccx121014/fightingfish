# 划拳小鱼版 - Fist Fight Fish Version

一款基于传统划拳的策略对战游戏，融合了资源管理、回合制战斗和心理博弈元素。

## 游戏目标
- 通过策略性出招击败对手
- 合理管理子弹资源和特殊技能使用次数
- 在不同游戏模式下提升个人积分

## 核心资源
- **子弹**：攻击招式的基础消耗资源
- **特殊技能次数**：单反、巨反、地雷、小鱼有使用限制

## 技能系统

### 攻击技能
- **上弹**：子弹+1，无消耗
- **单枪**：消耗1子弹，克制上弹、波防、核防；被普防、单反克制
- **双枪**：消耗2子弹，克制上弹、波防、单枪、核防、地雷；被普防、单反克制
- **波**：消耗3子弹，克制上弹、单枪、普防、单反、核防、地雷、双枪、小鱼；被波防克制
- **核弹**：消耗5子弹，克制上弹、单枪、普防、单反、波防、地雷、双枪、小鱼；被核防、巨反克制

### 防御技能
- **普防**：无消耗，克制单枪、双枪；被波克制
- **波防**：无消耗，克制波；被单枪、双枪克制
- **核防**：无消耗，克制核弹；被单枪、双枪克制

### 特殊技能
- **单反**：消耗1次，反弹单枪、双枪；可使用2次
- **巨反**：消耗1次，反弹单枪、双枪、波、核弹、地雷；可使用1次，第4回合后可用
- **地雷**：消耗1次，克制上弹、单枪；可使用2次，第4回合后可用
- **小鱼**：消耗1次，吸取对手所有子弹；可使用1次

## 技能互动规则
1. **反弹优先**：反弹技能（单反、巨反）优先于其他技能效果
2. **克制关系**：攻击克制防御或防御克制攻击
3. **互相抵消**：部分技能组合会互相抵消效果
4. **特殊效果**：小鱼的子弹吸取效果等

## 游戏模式
- **练习模式**：与AI对手练习，不影响积分
- **单人匹配**：与AI对手对战，影响积分
- **多人联机**：正在开发中

## 游戏流程
1. 登录/注册账号
2. 选择游戏模式
3. 回合制战斗（双方同时选择技能）
4. 技能效果结算
5. 战斗结束，查看结果

## 如何运行
1. 下载游戏文件到本地
2. 使用浏览器打开 `fistgame.html` 文件
3. 开始游戏

## 注意事项
- 为防止代码窃取，游戏禁用了右键菜单和Ctrl键
- 游戏数据保存在本地浏览器中

- 本地存储用户数据和排行榜
- 智能AI对手策略
