划拳小鱼版

游戏概述

划拳小鱼版是一款基于传统划拳的策略对战游戏，融合了资源管理、回合制战斗和心理博弈元素。玩家通过选择不同招式进行对战，目标是击败对手。

基础规则

游戏目标

• 通过策略性出招击败对手

• 合理管理子弹资源和特殊技能使用次数

• 在不同游戏模式下提升个人积分

核心资源

• 子弹：攻击招式的基础消耗资源

• 特殊技能次数：单反、巨反、地雷、小鱼有使用限制

完整技能系统

1. 攻击类技能

上弹
• 效果：子弹数量+1

• 消耗：0子弹

单枪
• 效果：杀死上弹、波防、核防

• 消耗：1子弹

• 被克：普防、单反

双枪
• 效果：杀死上弹、波防、单枪、核防、地雷

• 消耗：2子弹

• 被克：普防、单反

• 备注：多人联机中可以分成两个单枪打两个人

波
• 效果：杀死上弹、单枪、普防、单反、核防、地雷、双枪、小鱼

• 消耗：3子弹

• 被克：波防

核弹
• 效果：全局攻击，杀死除核防和巨反外的所有目标

• 消耗：5子弹

• 被克：核防、巨反


2. 防御类技能

普防
• 效果：防御单枪、双枪

• 消耗：0子弹

• 被克：波

波防
• 效果：防御波、小鱼，反弹地雷

• 消耗：0子弹

核防
• 效果：防御核弹和地雷

• 消耗：0子弹

3.特殊类技能

单反
• 效果：反弹全局单枪、双枪

• 限制：一局2次，不可连续使用

巨反
• 效果：反弹全局所有攻击（除上弹）

• 限制：一局1次，第4回合及以后使用

地雷
• 效果：杀死上弹、单枪

• 限制：一局2次，第4回合及以后使用，不可连续使用

• 被克：波防反弹，与核防、小鱼抵消

小鱼
• 效果：吸取对方所有子弹（若对方当前回合上弹，则一并吸走）

• 限制：一局1次

• 特殊机制：

  • 对方有巨反时：吸子弹者子弹清空，被吸者获得其子弹

  • 多人吸一人：子弹均分（向上取整）

  • 与单反、单枪、双枪、地雷互相抵消

  • 被波、核弹杀死

完整对战流程

回合结构

1. 准备阶段：查看当前资源状态
2. 决策阶段：选择本回合使用的招式
3. 执行阶段：双方招式同时生效
4. 结算阶段：处理招式互动和结果

招式互动规则

攻击与防御

• 直接击杀：攻击技能命中无防御的相克目标

• 完全防御：防御技能成功抵挡相克攻击

• 互相抵消：部分技能相遇时双方无效

反弹机制

• 单反：反弹单枪、双枪

• 巨反：反弹除上弹外的所有攻击

• 波防：反弹地雷

• 反弹的攻击会作用于原使用者

游戏模式

1. 练习模式

• 对手：AI（简单/普通/困难难度）

• 目的：熟悉游戏机制和技能搭配

• 积分：无积分变化

2. 单人匹配

• 对手：真实玩家或AI（基于积分匹配）

• 匹配规则：选择积分最接近的对手

• 积分变化：

  • 获胜：+2分

  • 失败：-1分（不会低于0分）

3. 多人联机

• 房间制：创建/加入房间

• 人数：支持2人以上对战

• 积分变化：

  • 获胜：+人数×2分

  • 失败：-1分


游戏界面与操作

主要界面

1. 登录/注册：用户账户管理
2. 主菜单：模式选择、排行榜查看
3. 游戏界面：实时对战显示
4. 结果界面：战斗数据统计

操作流程

1. 选择游戏模式
2. 匹配对手（单人/多人模式）
3. 回合制出招选择
4. 自动结算战斗结果
5. 查看积分变化和战绩
